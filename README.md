# Visualizacoes do Programa Melhor em Casa

Este repositorio contem apenas o site estatico e as visualizacoes finais da
Analise Exploratoria de Dados (AED) do projeto de Iniciacao Cientifica
**Otimizacao de Rotas de Atencao Domiciliar**.

O processamento dos dados, os scripts de analise e a geracao das visualizacoes
foram realizados no repositorio principal do projeto:

**Visualizacao-Programa-Melhor-em-Casa**  
https://github.com/FerSuaiden/Fapesp---Otimiza-o-de-Rotas-Atendimento-Domiciliar

As paginas, graficos, mapas e arquivos publicados neste repositorio sao
resultados derivados desse processamento. Portanto, este repositorio funciona
como uma versao de apresentacao/publicacao do site, sem incluir o codigo
completo usado para tratar os dados.

## Conteudo

- Paginas HTML do site
- Arquivos CSS e JavaScript usados na interface
- Mapas interativos exportados em HTML
- Graficos e imagens finais das visualizacoes
- Arquivos auxiliares necessarios para navegacao local do site

## Escopo

Este repositorio nao contem o pipeline completo de processamento dos dados.
Ele deve ser usado para consulta, navegacao e publicacao das visualizacoes
geradas a partir do repositorio principal.

## Fontes de dados

As visualizacoes foram produzidas a partir de dados publicos do CNES/DATASUS,
da Classificacao Brasileira de Ocupacoes (CBO) e do IBGE, utilizados para
caracterizar as equipes de Atencao Domiciliar no Brasil.

## Como visualizar

O site publicado esta disponivel em:

https://otimhomecare.icmc.usp.br/

## Atualizacao das visualizacoes

Para atualizar o conteudo deste repositorio, as analises devem ser refeitas no
repositorio principal. Em seguida, os arquivos finais gerados devem ser
copiados para este repositorio de publicacao.
